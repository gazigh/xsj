# 概要
**安全检测**模块是对当前流行的移动应用包进行安全扫描及分析. 由Django框架开发
# 主要功能

 1. **检测**, 对*目标文件*进行(静态)检测
 2. **报表**, 检测结果支持PDF格式导出
 3. **分析**, 对检测结果进行分析[**==漏洞占比, 漏洞分类, 漏洞排名, 应用分析排名==**]
